A web-based CRM dashboard designed for small agencies to efficiently manage leads, clients, and daily workflows. This project focuses on creating a scalable and intuitive interface that supports real-world agency operations.
The dashboard enables teams to track leads across multiple stages, manage active clients, and plan tasks with follow-up reminders, all in one place. It is designed with a workflow-first approach, ensuring smooth transitions from lead acquisition to client management.
Built with B2B SaaS UX principles, the interface prioritizes clarity, usability, and data organization, making it easy for teams to monitor client engagement and make informed decisions quickly.

# Features
* Lead tracking across multiple stages (New, Contacted, Converted)
* Client list management with clear status overview
* Visual lead pipeline for easy workflow tracking
* Detailed client profile with engagement insights
* Task management with follow-up reminders
* Clean and intuitive dashboard navigation
* Scalable layout for handling large datasets
* Workflow-focused design for daily agency operations
